功能 2：USER 角色的表單顯示方式

USER 只能編輯「測試報告」與「Completed / Delayed 狀態」，其他欄位不可改。請問不可編輯的欄位應如何呈現？

A：灰底顯示（Disabled）

不可編輯的欄位仍顯示，但呈灰底、游標禁止，USER 可以看到所有資料，只是無法修改。

ℹ️ 以 USER 身份編輯，僅可修改測試報告與排程狀態

http://report-link...

B：僅顯示可編輯欄位

USER 開啟表單時，只呈現「測試報告」及「Completed / Delayed」，其他欄位完全不顯示，版面更簡潔。
