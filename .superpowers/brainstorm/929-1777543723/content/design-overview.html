功能 1 — Tooltip 智慧定位

觸發點：GanttChart.tsx 的 onMouseEnter
邏輯：在 tooltip state 中同時記錄滑鼠座標，render 時計算：
  • 若游標距右側邊框 < 220px → tooltip 向左彈出（offset 改為負值）
  • 若游標距底部邊框 < 180px → tooltip 向上彈出
  • 其餘情況維持預設（右下偏移 +14px）
📁 修改檔案：src/components/schedule/GanttChart.tsx

功能 2 — USER 角色欄位限制

✅ USER 可編輯
  • 測試報告（testReport）
  • Completed（isCompleted）
  • Delayed（isDelayed）
  • 延遲原因（delayReason）*勾選Delayed才出現
🚫 USER 唯讀（灰底 disabled）
  • 工作類別、專案名稱、工作內容
  • 測試單位、測試人員
  • 時間資源、需求人員
  • 起始日期、完成日期
📁 修改檔案：src/components/schedule/ScheduleFormModal.tsx

功能 3 — 甘特圖左欄拖曳調整

工作排程欄
可拖曳
⋮
時間軸區域
  • 把手寬度 6px，hover 時變藍色，cursor: col-resize
  • 寬度範圍：最小 180px ～ 最大 480px，預設 248px
  • 拖曳結束後寫入 sessionStorage('ganttLeftWidth')

  • 重新登入後 sessionStorage 清除 → 自動回到 248px
📁 修改檔案：src/components/schedule/GanttChart.tsx

功能 4 — Excel 覆蓋差異確認對話框

觸發條件：mode = 'replace' 且點擊「確認匯入」
流程：
  1. 用 taskDescription 匹配現有排程與 Excel 匯入資料
  2. 比較四個欄位：testReport、isCompleted、isDelayed、delayReason
  3. 若有任何差異 → 彈出獨立確認 modal，列出差異表格（專案名稱 + 欄位 + 舊值 + 新值）
  4. 若無差異（或無配對記錄） → 直接執行覆蓋，不彈出
ℹ️ 全新記錄（Excel 有但現有無）不納入差異比較，直接新增
📁 修改檔案：src/components/schedule/ExcelImportModal.tsx（新增 DiffConfirmModal 元件）
